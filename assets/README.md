# Assets - Diagramas e Imagens

Esta pasta contem todos os diagramas ilustrativos e imagens usadas nos notebooks.

## Estrutura

Todos os assets ficam na raiz desta pasta, sem subdiretorios.

## Padroes de Nomenclatura

Formato: `[categoria]-[algoritmo]-[tipo].png`

**Exemplos:**
- `sorting-bubble-sort-overview.png`
- `sorting-bubble-sort-steps.png`
- `sorting-comparison.png`
- `searching-binary-search-steps.png`
- `graphs-bfs-visualization.png`
- `dp-fibonacci-table.png`

## Tipos de Diagrama:

- `overview`: Visao geral do algoritmo
- `steps`: Passos detalhados
- `comparison`: Comparacao entre algoritmos
- `visualization`: Visualizacao completa
- `table`: Tabelas DP

## Ferramentas

Consulte `config/studies/programming-kb/GUIA_DIAGRAMAS.md` para detalhes sobre:
- Excalidraw (principal)
- Mermaid (secundaria)
- IAs generativas (opcional)
- Script Python automatizado (`scripts/diagram_generator.py`)

## Estilo Visual

- Fundo branco ou creme claro
- Cores primarias (azul, vermelho, verde)
- Linhas limpas e claras
- Texto legivel (fonte sans-serif, >= 12pt)
- Alta resolucao (minimo 1200px largura)
